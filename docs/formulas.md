# Formeln

- Haversine-Distanz: $d = 2R\arcsin(\sqrt{\sin^2(\Delta\phi/2) + \cos\phi_1\cos\phi_2\sin^2(\Delta\lambda/2)})$
- Geschwindigkeit: $v = s / \Delta t$
- Beschleunigung: $a = \Delta v / \Delta t$
- Steigungswinkel: $\theta = \arctan(\Delta h / d_h)$
- Antriebsleistung: $P = F \cdot v$
- Motordrehmoment: $T_m = T_w / i$
- Motorstrom: $I = T_m / k_t$
